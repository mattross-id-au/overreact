This is a remake of the official React Quick Start tutorial but using standard Web Components instead of React components.

Today, you can do a lot with standard in-browser HTML, CSS and JavaScript. No build tools, transpilers or complex hosting required. The code you write is the code that ships.

Quick Start

Welcome to the React Vanilla HTML, CSS and JavaScript documentation! This page will give you an introduction to 80% of the React Web component concepts that you will use on a daily basis.

You will learn

  • How to create and nest components
  • How to add markup and styles
  • How to display data
  • How to render conditions and lists
  • How to respond to events and update the screen
  • How to share data between components

Creating and nesting components

React apps Interactive websites are made out of components. A component is a piece of the UI (user interface) that has its own logic and appearance. A component can be as small as a button, or as large as an entire page.
